# adventure

## setup

```sh
$ sudo gem install dxopal
```

## run and debug

```sh
$ dxopal server
```

## build

```sh
$ dxopal update
```

## publish

1. GitHub Settings → Pages → select 'main' branch
2. access (repository name).github.io/adventure/
